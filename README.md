# dry-aged-deps

A CLI tool to list outdated npm dependencies and display how long they have been outdated, helping maintainers identify stale packages.

## Installation

```sh
npm install -g dry-aged-deps
```

## Usage

```sh
dry-aged-deps
dry-aged-deps --help
```
